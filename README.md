# iris-statistical-analysis
Exploratory statistical analysis of the Iris dataset using Microsoft Excel.
Iris Statistical Analysis

📊 Project Overview

This project performs statistical analysis on the famous Iris dataset using Microsoft Excel.
The goal of the project is to explore the dataset, calculate statistical measures, and visualize patterns in the data.

The Iris dataset is widely used in statistics and machine learning for data analysis and classification tasks.

---

📁 Project Files

- iris_statistical_analysis.xlsx – Excel workbook containing the dataset and statistical analysis.

---

🔍 Analysis Performed

The following statistical analyses were performed:

- Mean calculation
- Median calculation
- Standard deviation
- Minimum and maximum values
- Data comparison across features
- Basic data visualization

The analysis focuses on the main features of the dataset:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

🛠 Tools Used

- Microsoft Excel
- Basic statistical methods

---

📚 Dataset Information

The dataset used in this project is the Iris dataset, originally introduced by Ronald A. Fisher in 1936.
It contains measurements of iris flowers from three species:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

Each sample includes four features describing flower measurements.
